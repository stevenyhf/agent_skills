# 中国公路运输价格指数

基于公开数据和行业模型，为用户提供公路运输参考运价。

## 快速使用

用户输入：起始城市、目的城市、车型、季节 → 输出参考运价（元/吨公里 或 元/车）。

## 运价计算流程

### 第1步：识别输入参数

从用户消息中提取：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| origin | 起始城市 | 必填 |
| destination | 目的城市 | 必填 |
| vehicle_type | 车型 | 整车(标准厢式17.5m) |
| season | 淡旺季 | 根据当前月份自动判断 |
| cargo_type | 货物类型 | 标准普货 |
| weight | 货物重量(吨) | 按车型满载计算 |

### 第2步：计算基准距离

使用内置城市坐标库（`data/city_distances.json`）查询城市间公路运距。如城市不在库中，使用 Haversine 公式计算直线距离后乘以 1.35 折算为公路里程。

### 第3步：确定车型参数

从 `data/vehicle_types.json` 读取车型参数：

| 车型代码 | 车型名称 | 额定载重(吨) | 车厢容积(m³) | 基准运价(元/吨公里) |
|----------|----------|-------------|-------------|-------------------|
| FT-17T | 整车 17.5米厢式 | 30 | 130 | 0.35 |
| FT-13T | 整车 13米半挂 | 25 | 90 | 0.38 |
| FT-9T | 整车 9.6米厢式 | 15 | 55 | 0.42 |
| FT-7T | 整车 7.8米厢式 | 10 | 40 | 0.48 |
| FT-4T | 整车 4.2米厢式 | 2 | 18 | 0.65 |
| LT-H | 零担重货 | — | — | 0.55 |
| LT-L | 零担轻货 | — | — | 0.85 |

### 第4步：应用季节系数

```python
# 季节系数 (1月=1)
SEASON_FACTORS = {
    "peak": [1, 2, 9, 10, 11, 12],    # 旺季月份
    "shoulder": [3, 4, 5, 6, 7, 8],     # 平季月份
}
SEASON_MULTIPLIER = {
    "peak": 1.12,       # 旺季上浮12%
    "shoulder": 1.00,   # 平季基准
    "low": 0.90,        # 淡季下浮10%
}
```

**细化月度系数：**

| 月份 | 系数 | 说明 |
|------|------|------|
| 1月 | 1.15 | 春节前旺季，运力紧张 |
| 2月 | 0.88 | 春节假期，需求骤降 |
| 3月 | 0.95 | 复工复产过渡期 |
| 4月 | 1.00 | 正常水平 |
| 5月 | 1.02 | 略有回升 |
| 6月 | 0.97 | 雨季影响部分线路 |
| 7月 | 0.95 | 夏季淡季 |
| 8月 | 0.93 | 夏季最淡 |
| 9月 | 1.05 | 金九银十备货 |
| 10月 | 1.08 | 国庆后需求集中 |
| 11月 | 1.15 | 双十一/年终备货 |
| 12月 | 1.12 | 年终赶工 |

### 第5步：应用区域系数

不同区域的运价水平存在差异（CFLP指数9大区域）：

| 区域 | 覆盖省份/城市 | 区域系数 |
|------|--------------|---------|
| 东北 | 辽宁、吉林、黑龙江 | 1.08 |
| 华北 | 北京、天津、河北、山西、内蒙古 | 0.95 |
| 华东 | 上海、江苏、浙江、安徽、山东 | 0.92 |
| 华中 | 河南、湖北、湖南、江西 | 0.97 |
| 华南 | 广东、广西、福建、海南 | 1.02 |
| 西南 | 四川、重庆、贵州、云南、西藏 | 1.12 |
| 西北 | 陕西、甘肃、青海、宁夏、新疆 | 1.15 |
| 港澳台 | — | 特殊处理 |

**跨区域运输系数 = (出发地系数 + 目的地系数) / 2**

### 第6步：参考指数校准

基于 CFLP-林安中国公路物流运价指数进行校准：

- **基期**：2012年12月 = 100（月指数）
- **2025年指数均值**：约 105 点
- **2026年2月指数**：105.3 点
- **校准系数** = 当前指数 / 100 = 1.053

> 指数会随时间变化，使用 `data/index_history.json` 中的最新值。如无最新数据，使用 105.3 作为默认校准系数。

### 第7步：计算最终运价

```
整车运价 = 基准单价(元/吨公里) × 距离(公里) × 区域系数 × 季节系数 × 指数校准系数 × 载重(吨)

零担运价 = 基准单价(元/吨公里) × 货物重量(吨) × 距离(公里) × 区域系数 × 季节系数 × 指数校准系数
```

**零担最低收费**：按件计费的起步价为 30-50 元（根据区域），不满起步价按起步价收取。
