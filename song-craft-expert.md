# 音乐词曲创作专家 · Song Craft Expert

## 概述

本 Skill 是一个端到端的音乐创作系统，覆盖从歌词创作、旋律作曲、专业记谱到语音合成的完整流程。支持四种核心风格和全链路修改。

**四位风格锚点：**
- 🖊️ **作词**：方文山（中国风词宗，画面感极强）
- 🎹 **作曲**：林俊杰（旋律大师，情绪张力极强）
- 🎤 **演唱**：周深（空灵声线，男女声自由切换）
- 🎼 **记谱**：简谱（数字谱）+ 五线谱（LilyPond 渲染）

**技术栈：**
- `music21` — 作曲与五线谱 MIDI 生成
- `lilypond` — 五线谱 PDF/PNG 渲染
- `edge-tts`（微软开源）— TTS 语音合成
- `pydub` + `ffmpeg` — 音频后处理
- `numpy` — 信号处理基础

## 四项核心功能

### 功能一：作词（方文山风格）

**方文山词风六要素：**

1. **画面感极强** — 每一句歌词就是一幅画面，像电影镜头："天青色等烟雨，而我在等你"
2. **中国风意象** — 善用古典意象入现代歌词：青瓷、宣纸、烟雨、古道、长亭、琵琶
3. **押韵精妙** — 韵脚工整，转韵自然，一韵到底或 ABAB 交替
4. **文字游戏** — 善用对仗、叠字、谐音双关："雨纷纷，旧故里草木深"
5. **情感留白** — 不直说情感，用场景和物件暗示："我送你离开，千里之外"
6. **叙事结构** — 歌词有完整的故事线索，主歌铺垫场景，副歌爆发情感

**歌曲结构模板：**

```
[主歌A1] 4行 — 场景铺陈（远景→近景）
[主歌A2] 4行 — 情绪铺垫（外景→内心）
[副歌B1] 4行 — 情感爆发（核心记忆点/hook）
[主歌A3] 4行 — 故事推进
[副歌B2] 4行 — 情感递进
[桥段C]  2-4行 — 情绪转折或升华
[副歌B3] 4行 — 最终情感高潮（可加变化）
[尾奏]   2行 — 余韵/留白
```

**作词输出格式：**

```markdown
# 《歌名》

## 元信息
- **词**：方文山风格
- **曲**：林俊杰风格
- **调式**：C大调（或指定调式）
- **拍号**：4/4
- **BPM**：72-80（林俊杰慢歌）/ 90-110（中速）
- **韵脚**：ang / i / en（具体韵部）

## 歌词正文

### [主歌A1]
第一行
第二行
第三行
第四行

### [副歌B1]
（歌词内容）

## 创作说明
- 意象选取：说明使用了哪些核心意象
- 情感线索：从X到Y的情感弧线
- 韵脚设计：哪几行押什么韵
```

### 功能二：作曲（林俊杰风格）

**林俊杰旋律六要素：**

1. **五声音阶为骨架** — 大量使用 do-re-mi-so-la（1-2-3-5-6），偶尔加入 fa/si 制造张力
2. **旋律弧线优美** — 主歌平稳叙事（中低音区），副歌大幅跳跃（高音区），桥段情绪转折
3. **和弦进行讲究** — 常用进行：
   - 抒情：I - V - vi - iii - IV - I - IV - V（卡农变体）
   - 情感爆发：vi - IV - I - V（副歌常用）
   - 林俊杰特色：vi - IV - V - iii（带有小调忧郁感）
4. **Hook 记忆点强** — 副歌第一句必须有强记忆点（"修炼爱情的心酸""一千年以后"）
5. **节奏与歌词紧密咬合** — 一字一音为主，关键情感词拉长旋律
6. **桥段升华** — 常用转调或旋律变化制造情感升华

**作曲参数：**

| 参数 | 抒情慢歌 | 中速情歌 | 轻快小调 |
|------|---------|---------|---------|
| BPM | 68-78 | 88-100 | 110-130 |
| 调式 | C/G大调 | D/E大调 | F/A大调 |
| 拍号 | 4/4 | 4/4 | 4/4 或 6/8 |
| 音域 | 中C到高C | 低音A到高音D | 中音区 |

**作曲输出：** 调用 `scripts/compose_melody.py` 生成旋律数据，再由记谱脚本渲染。

### 功能三：记谱（简谱 + 五线谱）

#### 3a. 简谱（Numbered Notation）

调用 `scripts/jianpu_generator.py`，输出格式：

```
1 = C  4/4  ♩= 76

[主歌A1]
| 5  3  5  6 | 5  -  -  - |
  天  青  色    等
| 3  2  1  2 | 3  -  2  - |
  烟  雨  而  我  在  等  你
| ...

[副歌B1]
| 1̇  6  5  3 | 5  6  1̇  - |
  （高音1上方加点）
```

**简谱标记规则：**
- `1 2 3 4 5 6 7` = do re mi fa sol la si
- `1̇` (上方点) = 高音；`₁` (下方点) = 低音
- `-` = 延长一拍；`_` (下划线) = 半拍
| `0` = 休止符
| `|` = 小节线

#### 3b. 五线谱（Staff Notation）

调用 `scripts/sheet_music_writer.py`，通过 LilyPond 渲染为 PDF/PNG：

- 包含：调号、拍号、音符、歌词对齐
- 输出：`{歌曲名}_五线谱.pdf` 和 `{歌曲名}_五线谱.png`

---

### 功能四：TTS 演唱合成

调用 `scripts/sing_synthesizer.py`，使用微软 Edge TTS：

**声音选择：**

| 歌手风格 | 推荐TTS声音 | Edge TTS Voice ID |
|---------|-------------|-------------------|
| 周深（空灵男声） | 男声 | `zh-CN-YunxiNeural` |
| 周深（空灵女声） | 女声 | `zh-CN-XiaoxiaoNeural` |
| 其他男歌手 | 男声 | `zh-CN-YunxiNeural` |
| 其他女歌手 | 女声 | `zh-CN-XiaoyiNeural` |

**输出：**
- `{歌曲名}_演唱_{声音}.mp3` — 带歌词的 TTS 朗读
- `{歌曲名}_旋律.mp3` — MIDI 转音频的旋律参考

**音频后处理：**
- 使用 `pydub` 调整语速（匹配 BPM）
- 可选：叠加旋律音轨作为背景参考

## 完整工作流程

### 一键创作流程

```
用户输入：题材/主题/情感
          ↓
    ┌─ Step 1: 作词 ─┐
    │  方文山风格歌词   │
    └────────┬────────┘
             ↓
    ┌─ Step 2: 作曲 ─┐
    │  林俊杰风格旋律   │
    │  → MIDI 文件     │
    └────────┬────────┘
             ↓
    ┌─ Step 3: 记谱 ─┐
    │  简谱 TXT        │
    │  五线谱 PDF/PNG   │
    └────────┬────────┘
             ↓
    ┌─ Step 4: 演唱 ─┐
    │  TTS MP3         │
    │  旋律参考 MP3     │
    └────────┬────────┘
             ↓
    ┌─ Step 5: 交付 ─┐
    │  全部保存到       │
    │  ~/文档/          │
    └─────────────────┘
```

### 修改流程

用户可对任何环节提出修改要求：
- "副歌韵脚换个更押韵的" → 只重新生成歌词副歌部分
- "旋律太高了，降一个调" → 重新生成旋律 + 记谱
- "换成女声唱" → 切换 TTS 声音重新合成
- "整体慢一点" → 调整 BPM，全部重新生成

每次创作会在 `~/文档/` 下生成一个歌曲目录：

```
~/文档/{歌曲名}/
```

## 脚本说明

| 脚本 | 功能 | 依赖 |
|------|------|------|
| `scripts/jianpu_generator.py` | 生成简谱 TXT | numpy |

## 使用示例

### 示例1：中国风慢歌

```
用户：帮我写一首关于"烟雨江南"的歌

→ 作词：方文山式画面感歌词，韵脚 ang
→ 作曲：C大调抒情，BPM 72，卡农和弦进行
→ 记谱：简谱 + 五线谱
→ 演唱：周深风格（男声 YunxiNeural）
→ 输出：~/文档/烟雨江南/ 完整目录
```

### 示例2：都市情歌

```
用户：写一首关于深夜加班后思念恋人的歌

→ 作词：现代都市场景 + 古典情感留白
→ 作曲：D大调中速，BPM 88，情感爆发副歌
→ 演唱：女声（XiaoxiaoNeural）
→ 输出：~/文档/深夜思念/ 完整目录
```

### 示例3：修改

```
用户：这首歌副歌太高了，降两个key，换女声唱

→ 调整调式：D大调 → C大调
→ 重新生成旋律 + 简谱 + 五线谱
→ 切换声音：XiaoxiaoNeural
→ 输出：更新 ~/文档/ 下文件
```

## 风格边界

- ❌ 不做说唱/嘻哈风格（方文山/林俊杰/周深三人均非此风格）
- ❌ 不做英文歌词（方文山以中文词见长）
- ❌ 不用 AI 歌声合成替代 TTS（本系统基于微软开源 TTS，非商业歌声合成）
- ❌ 不生成伴奏/编曲（本系统仅覆盖旋律线，不含完整编曲）
- ❌ 不涉及版权音乐翻唱（仅原创词曲）
