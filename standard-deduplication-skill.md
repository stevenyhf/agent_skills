# 标准立项前去重及参考标准查询技能

本技能用于在拟制定新标准之前，通过全国标准公共信息服务平台进行标准去重和参考标准查询，避免重复制定已有标准，并提供相关标准参考。

## 触发条件

使用此技能当：
- 用户需要查询某个标准是否已被制定过
- 用户想知道是否有同名标准存在
- 用户希望了解有哪些类似标准可供参考
- 用户在标准立项前需要进行去重检查

不使用此技能当：
- 用户只是询问一般性的标准信息而不需要去重
- 用户需要查询其他国家的标准数据库
- 用户只需要简单的标准号查询而非全面去重

## 前置条件

- 需要能够访问全国标准公共信息服务平台（https://std.samr.gov.cn/）
- 用户提供拟新制定标准的全名
- 用户提供拟新制定标准名称的核心词语（用于相关标准检索）
- 指定工作夹路径用于保存检索结果文档

## 工作流程

### 步骤1：同名标准检索

1. 打开全国标准公共信息服务平台（https://std.samr.gov.cn/）
2. 在搜索框中输入用户提供的**拟新制定标准全名**
3. 点击检索按钮执行搜索
4. 分析搜索结果：
   - **如果有同名标准**：
     - 收集所有同名标准的网址
     - 提取每个标准的核心内容（标准号、标准名称、标准类别、发布状态等）
     - 按条整理输出到对话框
   - **如果没有同名标准**：
     - 在对话框中输出："无同名标准"

### 步骤2：相关标准检索

1. 在同一平台搜索框中输入用户提供的**拟新制定标准名称的核心词语**
2. 点击检索按钮执行搜索
3. 分析搜索结果：
   - **如果有相关标准**：
     - 收集所有相关标准的网址
     - 提取每个标准的核心内容（标准号、标准名称、标准类别、发布状态等）
     - 明确指出每个标准的类别（国家标准、行业标准、地方标准、团体标准、企业标准、国际标准）
     - 按条整理输出到对话框
   - **如果没有相关标准**：
     - 在对话框中输出："无相关标准"

### 步骤3：结果整理与文档生成

1. 将所有检索结果（同名标准检索结果 + 相关标准检索结果）整合
2. 生成本地文档，包含：
   - 检索时间
   - 拟新制定标准全名
   - 拟新制定标准核心词语
   - 同名标准检索结果（含标准网址和核心内容）
   - 相关标准检索结果（含标准类别、网址和核心内容）
3. 将文档保存到用户指定的工作夹中
4. 告知用户任务已完成，并提供输出结果摘要

## 输出合同

必须产生：
- 对话框中的检索结果输出（同名标准和相关标准）
- 本地文档文件（包含完整检索结果）
- 任务完成通知

## 异常处理

### 网站访问失败
- 如果无法访问全国标准公共信息服务平台，告知用户网络问题或网站不可用
- 建议用户稍后重试或提供替代方案

### 搜索结果为空或页面加载失败
- 如果搜索后页面无结果或加载失败，尝试重新搜索
- 如果多次失败，告知用户可能的原因（如关键词过于特殊、网站临时故障等）

### 缺少必要输入
- 如果用户未提供标准全名或核心词语，询问用户补充
- 如果用户未指定工作夹路径，询问用户或使用默认路径

### 标准类别识别
- 在检索结果中明确标注每个标准的类别
- 标准类别包括：国家标准、行业标准、地方标准、团体标准、企业标准、国际标准

## 注意事项

1. **不分标准类别检索**：检索时不区分标准类别，但结果中需明确指出每个标准的类别
2. **核心内容提取**：对于每个找到的标准，应提取关键信息如标准号、标准名称、发布状态、实施日期等
3. **文档格式**：生成的本地文档建议使用Markdown或TXT格式，便于阅读和后续处理
4. **结果准确性**：确保检索结果的准确性，如有不确定的地方应向用户说明
