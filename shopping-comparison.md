# 购物比价 Skill

## 使用场景

用户想买某类商品，需要跨平台比价、筛选优质卖家、综合推荐。例如："我想买个500以内的蓝牙耳机，推荐一下"、"哪款空气炸锅性价比高"。

## 工作流

### 第一步：需求澄清

如果用户需求不够具体，先追问关键参数：

- **预算**：价格区间
- **用途**：主要使用场景（如：运动蓝牙耳机 vs 通勤降噪耳机）
- **偏好**：品牌倾向、功能偏好、是否介意非官方店
- **数量**：买一件还是多件

输出格式：
> 已明确需求：[复述用户需求]
> 预算：XXX
> 关键维度：[列举3-5个核心考量点]

### 第二步：遍历搜索（各平台）

对每个平台使用 `web_search` + 推荐搜索词进行抓取。**禁止**只搜一个平台就下结论。

**搜索策略：**
- 每个平台使用多个搜索词交叉验证，例如：`"2025 蓝牙耳机 销量排行"`、`"蓝牙耳机 京东 热卖"`、`"500元以内 蓝牙耳机 推荐"`
- 搜索来源：优先使用各平台自身搜索结果（`site:jd.com 蓝牙耳机 销量`），辅以第三方比价/评测网站（什么值得买、慢慢买等）

**平台清单：** 京东、淘宝/天猫、拼多多。如果商品品类适合，可补充：抖音商城、得物、唯品会、苏宁等。

### 第三步：数据提取与结构化

对每个搜索结果提取以下字段：

| 字段 | 说明 |
|------|------|
| 商品名称 | 完整名称 |
| 价格 | 当前售价（标价+券后价，如有） |
| 平台 | 所在平台 |
| 店铺 | 店铺名 + 店铺类型（官方旗舰店/专卖店/个人店） |
| 销量 | 月销量/累计销量（标注数据来源时间） |
| 好评率 | 好评率数值（标注样本量，如"5000+评价"） |
| 关键卖点 | 宣传的核心功能/卖点 |
| 负面信息 | 差评集中点、常见问题 |

**结构化要求：**
- 每个商品数据必须标注**数据来源URL**和**抓取时间**
- 数据**禁止PS/美化**，如实呈现

### 第四步：信息可靠性校验（关键步骤）

对收集到的每条信息做可信度评估：

**高可信来源：**
- 平台官方店铺页面（销量、评价数据）
- 官方旗舰店、品牌自营店
- 大型第三方评测机构的系统测试

**需交叉验证的来源：**
- 个人评测/开箱（可能恰饭）
- 社交媒体种草文（小红书、抖音）
- 非官方店铺的描述

**可靠性降级因素：**
- 销量数据来源不明/时间不明确 → 降级
- 好评率异常高（>98%）且评价数少（<200）→ 怀疑刷单
- 价格远低于市场均价 → 怀疑假货/翻新
- 店铺为新开店或评分异常低 → 谨慎

**交叉验证方法：**
- 将 A 平台的信息到 B 平台验证
- 用第三方比价平台查看历史价格走势
- 查品牌官方资料确认规格参数

**外部引用规则：**
- 第三方评测：优先引用**垂直媒体**（如ZOL、中关村在线、太平洋电脑网）和**机构评测**
- KOL评测：标注"博主合作/赞助"标签（如有赞助必须注明）
- 用户评价：引用有代表性的真实评价，标注"个体体验，非普遍结论"
- 历史价格：引用慢慢买、什么值得买等比价平台的历史价格曲线

### 第五步：按平台输出前三推荐

按平台输出，格式示例：

```
━━━ 京东 TOP 3 ━━━

1️⃣ [商品名]
   💰 ¥XXX（券后价 ¥XXX）
   🏪 店铺：XXX官方旗舰店
   📊 销量：10万+ | 好评率：97%（10万+评价）
   ✅ 核心优势：XXX
   ⚠️ 注意：XXX
   🔗 [商品链接](URL)
   📋 数据时间：2025-XX-XX

2️⃣ ...
3️⃣ ...

━━━ 淘宝/天猫 TOP 3 ━━━
...
━━━ 拼多多 TOP 3 ━━━
...
```

### 第六步：跨平台综合分析

对比维度包括：

- **价格**：各平台同款/同类最低价
- **售后**：各平台退换货政策差异
- **物流**：物流速度和服务质量
- **正品保障**：平台对该品类的正品保障力度
- **促销活动**：当前各平台的促销/满减

### 第七步：提出最终推荐名单

**原则：认真鉴别，慎重提出**

**最终推荐名单结构：**

```
╔══════════════════════════════════════╗
║        📋 最终购物推荐名单           ║
╚══════════════════════════════════════╝

🥇 首选推荐：[商品名]
   平台/店铺：XXX
   推荐理由：[2-3句话]
   适合人群：[谁适合买]
   参考价格：¥XXX

🥈 性价比之选：[商品名]（可选）
🥉 品质之选：[商品名]（可选）

💡 购买建议：[一句话提示]
🤔 风险提示：[如果有的话]
```

**推荐约束规则：**
1. **没有完美商品** —— 必须在推荐中说明缺点/妥协点
2. **不确定性必须声明** —— 如果数据不全、价格波动大、信息有矛盾，必须明确标注
3. **不推荐三类商品**：
   - 疑似刷单/虚假宣传的商品
   - 店铺资质存疑的商品
   - 价格显著异常（过低或虚高无依据）的商品
4. **量化的、来源标注的数据优先于感受型描述**
5. **不确定时，选择保守** —— 宁可不推，不要错推

## 示例

用户："想买个300-500的入耳式降噪蓝牙耳机，主要通勤用"

期望输出长文，涵盖：
- 需求确认
- 3个平台的搜索结果
- 各平台TOP3商品对比
- 信息可靠性说明
- 最终3个推荐（可选档次不同）

## 禁止行为

- ❌ 不可以只查一个平台就给结论
- ❌ 不可以照抄商家宣传文案当事实
- ❌ 不可以忽略差评/负面信息
- ❌ 不可以为凑推荐数量而降低标准
- ❌ 不可以推荐没有明确数据支撑的商品
- ❌ 不能不标注数据来源和时间
- ❌ 不能使用"大家都说好""公认"等无依据的表述
