## 技能描述

每日自动抓取中国31个省市不知名城市或县的当地新闻，形成全国小城热点新闻汇集，整理成日报推送。展现小城风采，讲述温暖故事。

## 触发场景

当用户需要：
- 获取全国不知名城市的每日新闻汇总
- 了解小城的社会民生、文化活动、经济动态、旅游亮点
- 定期接收城市轮换的新闻日报
- 探索全国小众城市的地方故事

**触发词**：城市新闻日报、小城新闻、不知名城市新闻、城市热点汇集、每日城市报告

## 功能特性

- **全量覆盖**：31个省市，每省精选不知名城市，每3天轮换一次
- **四大内容方向**：
  - 社会民生（社区改造、便民措施）
  - 文化活动（非遗展示、节庆活动）
  - 经济动态（特色产业、乡村振兴）
  - 旅游亮点（小众景点、民俗体验）
- **深度解读**：每条新闻100字深度解读，不只标题
- **多源采集**：当地融媒体、日报晚报、主流新闻平台、社交媒体
- **智能轮换**：每3天自动更新城市名单，持续覆盖更多小城
- **定时推送**：每日凌晨6:00采集，上午8:00准时推送

## 信息来源优先级

### 第一优先级（零风险）
- 城市官网官方RSS订阅源
- 公开新闻API接口
- 省级新闻云平台标准接口
- 全国90%以上县市融媒体中心已对接标准接口

### 第二优先级（低风险）
- 政府网/融媒体中心官网
- 每2个请求间隔30秒
- 携带标准浏览器请求头
- 模拟真人浏览频率

### 第三优先级（兜底备份）
- 官方抖音账号
- 官方微博账号
- 官方微信公众号
- 社交平台公开内容

## 使用方法

### 首次全量采集

```bash
# 启动首次采集（今日18点前完成）
python3 city_report.py --init

# 指定推送时间
python3 city_report.py --init --push-time 20:00
```

### 日常自动化运行

```bash
# 每日凌晨6:00自动采集
python3 city_report.py --daily

# 手动触发采集
python3 city_report.py --manual

# 查看当前城市名单
python3 city_report.py --list-cities
```

### 城市轮换管理

```bash
# 查看当前轮换周期
python3 city_report.py --cycle-info

# 手动触发城市轮换
python3 city_report.py --rotate-cities

# 自定义轮换周期（默认3天）
python3 city_report.py --rotate-days 5
```

## 命令行选项

- `--init`：首次全量采集初始化
- `--daily`：日常自动化采集模式
- `--manual`：手动触发采集
- `--push-time`：指定推送时间（格式：HH:MM）
- `--list-cities`：查看当前城市名单
- `--cycle-info`：查看轮换周期信息
- `--rotate-cities`：手动触发城市轮换
- `--rotate-days`：自定义轮换周期天数

## 城市轮换机制

- **轮换周期**：每3天换一批城市
- **候选池**：每省3个候选城市
- **完整周期**：9天（每省3个城市 × 3天）
- **长期目标**：31省 × 31市 × 31条温暖故事

## 反爬安全保障机制

### 频率控制
- 单个官网每24小时仅发起2次访问
- 采集当日新闻 + 校验源站可用性
- 远低于普通用户访问频率

### 多源兜底
- 每个城市至少预留3个官方信息源
- 一个渠道异常自动切换下一个
- 确保内容不断更

### 合规底线
- 仅抓取官方公开可免费访问的新闻
- 严格遵守各网站Robots协议
- 不抓取任何非公开内容

- 新闻内容智能分类与摘要
